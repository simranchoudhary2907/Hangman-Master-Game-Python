# 🎮 Hangman Master Game

A console-based Hangman game built using Python where players guess a hidden word one letter at a time. The game includes multiple categories, difficulty levels, hints, achievements, score tracking, streak tracking, and ASCII Hangman visualization.

---

## 🚀 Features

* 🎯 Random word selection
* 📚 Multiple categories

  * Programming
  * Technology
  * Cyber Security
* 🎚️ Three difficulty levels

  * Easy
  * Medium
  * Hard
* 💡 Hint system
* ❤️ Limited lives system
* 🎨 ASCII Hangman visualization
* 🏆 Achievement system
* 📈 Score tracking
* 🔥 Winning streak tracker
* 🔄 Play again option
* ✅ Input validation

## 🎮 How to Play

1. Run the Python file.
2. Select a difficulty level.
3. A category and hidden word will be chosen randomly.
4. Guess one letter at a time.
5. Use hints if available.
6. Win by guessing the complete word before running out of lives.
